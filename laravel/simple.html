Simple Project

Kali ini akan dijelaskan contoh sederhana dalam aplikasi laravel sederhana, berikut tahapan-tahapannya

1. Controller

# Create

Yang pertama kita akan mencoba membuat 1 controller dengan nama SimpleController. Untuk membuatnya kita dapat menggunakan laravel artisan command line dengan menggunakan perintah sebagai berikut

php artisan make:controller SimpleController

File controller ini akan disimpan di dalam folder app/http/Controllers.

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class SimpleController extends Controller
{
//
}

# Function

Setelah SimpleController terbuat, selanjutnya kita akan membuat 1 function dengan nama index untuk menampilkan sebuah tulisan Halo Selamat Datang

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class SimpleController extends Controller
{
public function index()
{
return response('Halo Selamat Datang');
}
}

2. Route

Selanjutnya kita akan menentukan rute URL simple agar diarahkan ke function index pada SimpleController, sehingga tulisan Halo Selamat Datang dapat dilihat pada browser.

Anda dapat membuat rute URL tersebut pada file routes/web.php

<?php

use Illuminate\Support\Facades\Route;

// Load simple controller
use App\Http\Controllers\SimpleController;

// define route
Route::get('simple', [SimpleController::class, 'index']);

3. Result

Untuk melihat hasilnya, silahkan Anda buka aplikasi browser kemudian akses url http://localhost:8000/simple
