Login Project

Kali ini akan dijelaskan contoh pembuatan halaman login sederhana dalam laravel, berikut tahapannya

1. Controller

# Create

Yang pertama kita akan mencoba membuat 1 controller dengan nama LoginController. Untuk membuatnya kita dapat menggunakan laravel artisan command line dengan menggunakan perintah sebagai berikut

php artisan make:controller LoginController

File controller ini akan disimpan di dalam folder app/http/Controllers.

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class LoginController extends Controller
{
//
}

# Function

Setelah LoginController terbuat, selanjutnya kita akan membuat 1 function dengan nama index untuk menampilkan sebuah html login form.

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class LoginController extends Controller
{
public function index()
{
return view('login');
}
}

2. View

Kita akan membuat HTML login form dengan nama login sebagaimana yang sudah ditentukan di LoginController.

<form method="post" action="{{ route('getdata')}}" enctype="multipart/form-data">

<div class="mb-3">
<label for="exampleInputEmail1" class="form-label">Email address</label>
<input type="text" name="email" class="form-control" id="exampleInputEmail1">
<div>

<div class="mb-3">
<label for="exampleInputPassword1" class="form-label">Password</label>
<input type="password" name="password" class="form-control" id="exampleInputPassword1">
</div>
<button type="submit" class="btn btn-primary">Submit<button>

</form>

3. Route

Selanjutnya kita akan menentukan rute URL login agar diarahkan ke function index pada LoginController.
Dan route getdata untuk menerima data dari form login nantinya.

Anda dapat membuat rute URL tersebut pada file routes/web.php

<?php

use Illuminate\Support\Facades\Route;

// Load login controller
use App\Http\Controllers\LoginController;

// define route
Route::get('login', [LoginController::class, 'index']);
Route::post('getdata', [LoginController::class, 'getData']);

4. Get Data

Di LoginController, kita tambahkan function getData untuk menerima data dari html login form.

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class LoginController extends Controller
{
public function index()
{
return view('login');
}

public function getData( Request $req )
{
// get email request data
$email = $req->input("email");

// get password request data
$password = $req->input("password");

return response('ok');
}
}

5. Validation

Selanjutkan kita mencoba untuk melakukan validasi data yang dikirim dari HTML form. Beberapa validasi yang akan dicoba adalah sebagai berikut,

  • Wajib diisi
  • Email format
  • Minimal 8 karakter

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class LoginController extends Controller
{
public function index()
{
return view('login');
}

public function getData( Request $req )
{
// s:validation
$req->validate([
'email' => 'required|email',
'password' => 'required|min:8',
]);
// e:validation

// get email request data
$email = $req->input("email");

// get password request data
$password = $req->input("password");

return response('ok');
}
}

6. Error Message

Untuk menampilkan pesan error dari validasi yang sudah dilakukan sebelumnya, dapat menambahkan beberapa code di login view yang sudah dibuat sebelumnya

<form method="post" action="{{ route('getdata')}}" enctype="multipart/form-data">

<div class="mb-3">
<label for="exampleInputEmail1" class="form-label">Email address</label>
<input type="text" name="email" class="form-control" id="exampleInputEmail1">
@error('email')
<div class="text-danger">{{ $message }}</div>
@enderror
<div>

<div class="mb-3">
<label for="exampleInputPassword1" class="form-label">Password</label>
<input type="password" name="password" class="form-control" id="exampleInputPassword1">
</div>
@error('password')
<div class="text-danger">{{ $message }}</div>
@enderror
<button type="submit" class="btn btn-primary">Submit<button>

</form>
