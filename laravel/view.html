View

View ini salah satu komponen dari laravel yang berfungsi untuk mendefinisikan file-file HTML yang akan digunakan dalam suatu project.

Ketentuan dalam pembuatan nama file view dalam laravel harus ditambahkan .blade.php sebagai akhirannya, seperti login.blade.php atau register.blade.php.

Dan secara default file view disimpan di dalam folder resources/views/.
