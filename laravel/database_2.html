Kemudian tambahkan function news baru di UserModel untuk menghubungkan dengan NewsModel
Function ini sebagai alias untuk mengambil data dari UserModel

<?php

namespace App\Models;

use App\Models\News;

class User extends Authenticatable
{
public function news()
{
return $this->hasMany( News::class );
}
}

Contoh penggunaan,

<?php

namespace App\Http\Controllers;

use App\Models\User;

class NewsController extends Controller
{
public function index()
{
$user = User::find(2);
$user_news = $user->news()->get();

foreach( $user_news as $news ){
echo $user->name.':'.$news->title.'<br>';
}
}
}

Jika ingin sebaliknya, ingin mengambil informasi user dari NewsModel, tambahkan function user seperti berikut

<?php

namespace App\Models;

use App\Models\User;

class News extends Model
{
public function admin()
{
return $this->belongsTo( User::class );
}
}
