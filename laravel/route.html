Route

Route berfungsi untuk mengatur lalu lintas request dari user dengan aplikasi laravel yang kita buat, apakah akan diarahkan ke controller atau langsung ke view.

Semua route di dalam suatu aplikasi laravel didefinisikan di dalam file routes/web.php. Rute yang telah didefinisikan dalam diakses dengan memasukkan rute URL pada browser Anda.

# Basic Route

Berikut contoh pembuatan rute hello

<?php

use Illuminate\Support\Facades\Route;

Route::get('/hello', function(){
return response('hello world');
});

Berikut contoh pembuatan rute welcome diarahkan ke controller

<?php

use Illuminate\Support\Facades\Route;
use App\Http\Controllers\WelcomeController;

Route::get('/welcome', [WelcomeController::class, 'index']);

# Parameter

Anda dapat menambahkan parameter ke dalam rute URL yang anda tentukan. Penambahan parameter ini terdapat 2 jenis,

1. Required parameter

<?php

use Illuminate\Support\Facades\Route;

Route::get('/user/{id}', function($id){
return response('ini user ke '.$id);
});

# controller
Route::get('/user/{id}', [UserController::class, 'index']);

2. Optional parameter

<?php

use Illuminate\Support\Facades\Route;

Route::get('/user/admin/{name?}', function($name = 'administrator'){
return response('Halo '.$name);
});

# controller
Route::get('/user/admin/{name?}', [UserController::class, 'admin']);

# Regular Expression Constraints

Parameter yang ditambahkan ke dalam rute URL dapat Anda batasi dengan beberapa format where sebagai berikut,

# Number only
Route::get('/user/{id}', [UserController::class, 'index'])->where('id', '[0-9]+');
# or
Route::get('/user/{id}', [UserController::class, 'index'])->whereNumber('id');

# Alphabet only
Route::get('/user/admin/{name?}', [UserController::class, 'index'])->where('name', '[A-Za-z]+');
# or
Route::get('/user/admin/{name?}', [UserController::class, 'index'])->whereAlpha('name');

# Alphabet & Number only
Route::get('/user/admin/{name?}', [UserController::class, 'index'])->where('name', '[0-9A-Za-z]+');
# or
Route::get('/user/admin/{name?}', [UserController::class, 'index'])->whereAlphaNumeric('name');

# Prefix

Di laravel, Anda dapat menggabungkan beberapa rute URL yang memiliki prefix sama dikelompokan menjadi satu, misalkan

http://localhost:8000/admin/login
http://localhost:8000/admin/profile

Dalam pendefinisiannya Anda dapat menuliskan sebagai berikut

<?php

use Illuminate\Support\Facades\Route;

Route::prefix('admin')->group(function(){

Route::get('login', function(){
return response('admin login');
});

Route::get('profile', function(){
return response('admin profile');
});

});
