Requests

Laravel telah menyediakan object oriented class dengan nama Request atau Illuminate\Http\Request. Class ini digunakan untuk menerima data HTTP Request dari browser, seperti form input, cookies dan files.

# Access the requests

Dalam penggunakan class Requestterdapat 2 cara, yaitu

use Illuminate\Http\Request;

Atau

use Request;

Seperti contoh UserController berikut ini yang akan menerima data HTTP request id

<?php

namespace App\Http\Controllers;

# Load Request class
use Illuminate\Http\Request;

class UserController extends Controller
{
public function index(Request $request, $id)
{
return response('ID user :'.$id);
}
}

# Request Input

Pada bagian berikut akan dijelaskan beberapa metode untuk menerima request data yang dikirim oleh user.

# Retrieve input form HTML form

Menerima data dari form input HTML dengan nama email.

$email = $request->input('email');

# Retrieve input form query string

Menerima data dari query string yang terdapat di URL, seperti http://localhost:3000/laravel.php?q=laravel

$query = $request->query('q');

# Request Files

# Retrieve upload file

Pada bagian berikut akan dijelaskan metode untuk menerima request file yang diupload oleh user.

$file = $request->file('photo');

# File name & extensions

Untuk selanjutnya dengan metode berikut Anda dapat mendapatkan informasi nama & extension file yang diupload.

# file name
$name = $request->file->getClientOriginalName();

# file extension
$ext = $request->file->extension();
