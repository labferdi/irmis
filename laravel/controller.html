Controller

Controller umumnya untuk digunakan untuk menangani keseluruhan request dari aplikasi yang Anda buat. Sebagai contohnya UserController untuk menangani selutuh request yang berhubungan dengan data user seperti menampilkan, input, update dan delete.

Secara default file controller disimpan di dalam folder app/http/Controllers.

# Create

Dengan menggunakan laravel artisan command line Anda dapat membuat controller dengan menggunakan perintah sebegai berikut

php artisan make:controller UserController

# Basic

Untuk selanjutkan kita akan melihat UserController yang telah dibuat sebelumnya.

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class UserController extends Controller
{
public function index()
{
return response('view dari user controller');
}
}

Untuk selanjutnya tentukan rute URL /user pada file routes/web.php yang diarahkan ke UserController agar dapat diakses melalui browser

<?php

use Illuminate\Support\Facades\Route;
use App\Http\Controllers\UserController;

Route::get('/user', [UserController::class, 'index']);
