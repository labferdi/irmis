Setup

Untuk project saat ini Kita akan menggunakan laravel versi 8. Sebelum membuat project dengan laravel, pastikan komputer Anda sudah terinstal PHP & Composer.

# Installation

Dengan menggunakan command prompt di windows, ketik command berikut untuk membuat project laravel baru.

composer create-project laravel/laravel:^8.0 nama-folder

Setelah folder nama-folder selesai terbuat, dilanjutkan dengan mengetik command berikut untuk menjalankan laravel local development server.

cd nama-folder

php artisan serve

Selanjutkan Anda dapat mengakses url http://localhost:8000 pada browser untuk mengakses aplikasi laravel Anda yang telah terinstal.
